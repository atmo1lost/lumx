# security

## reporting a vulnerability

if you discover a security vulnerability, please report it privately instead of creating a public issue.

send a direct message to [@atmo1lost](https://github.com/atmo1lost) on one of the supported platforms and include as much detail as possible, such as:

- a description of the vulnerability
- steps to reproduce it
- the affected version (if known)
- any screenshots or a proof of concept (not that needed)

i will investigate the report as soon as possible and work to resolve any confirmed issues.

for legal information, see `legal.md`.
